Arquitectura de PodioTest

Ecosistema modular desacoplado en servicios independientes bajo estructura de monorepo.

Arquitectura Full Stack

Diseño de un ecosistema modular desacoplado en servicios independientes bajo una estructura de monorepo. Comunica de forma eficiente el Frontend (Angular 17 Standalone) con el Backend (Java 17 / Spring Boot 3) y diversos microservicios satélites que abstraen las distintas responsabilidades del dominio.

Seguridad & Servicios

Securización completa de todos los endpoints mediante Spring Security. El acceso se gestiona a través de un riguroso control basado en roles y permisos (RBAC). Se integra directamente con Resend para asegurar la automatización y entrega de los flujos de emails transaccionales.

Rendimiento con Redis

Implementación estratégica de Redis Cache para permitir una lectura hiper-eficiente y gestión de datos de extrema concurrencia para las tablas de ranking globales. Esta capa intermedia reduce al mínimo la carga de la base de datos transaccional y minimiza la latencia en las consultas.
