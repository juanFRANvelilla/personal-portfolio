Crear usuario

Si no dispones de un usuario desde esta pantalla puedes crearlo. Haciendo uso de la api de Twilio se mandará un código aleatorio al número de teléfono proporcionado para validar que eres el propietario de este. Dispones de 3 intentos para introducir el código.



Acceso a la aplicacion


Al loguearte de forma exitosa se obtiene el JWT Token que servirá para autentificarse dentro de la aplicación.

Menu Principal

Desde aquí tenemos una vista rápida sobre nuestro balance indicando con un gráfico la cantidad que debemos y que nos deben, también podemos ver las notificaciones pendientes.

Contactos

Pantalla desde la que podemos ver nuestros contactos actuales y agregar nuevos. Al mandar solicitud de contacto, se le mostrará como una notificacion al entrar en la app.


Crear deuda

Desde aquí podemos crear una deuda de forma sencilla con cualquier contacto que tengamos.
